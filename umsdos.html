UMSDOS muLinux: self-extracting zip archive

In the case you experienced problems performing the standard installation steps (low memory, etc), a self-extracting zip is available, the image of a bare-minimum, pre-installed UMSDOS muLinux: the base system, with the USR, WKS and SRV addons already loaded (6MB circa).
The self-extracting archive expands itself in c:\linux and runs in the Dos prompt of your Win system: 

c:\> mu13r2.exe

  UMSDOS muLinux: zip archive

For pure DOS computers, you can use the bare zip archive 

c:\> pkunzip mu13r2.zip

Run it

As usual, if your system supports loadlin.exe, you can run muLinux from Win/DOS with: 

c:\> cd c:\linux
c:\> linux.bat
